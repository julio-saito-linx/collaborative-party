# Collaborative Party


## Run

1. `npm install`
2. `azk restart && azk logs -f`
3. `npm start`

----------------

## URLs

- http://localhost:3000/
- http://rethink-db.dev.azk.io/
- http://rethink-server.dev.azk.io/

----------------

## Build for production

`npm run build`

----------------

## Test

`(cd domain && npm run watch:test)`

----------------
